# Control: Shuffled Surprisal

## Purpose

Check whether the signal depends on the real centered-surprisal direction rather than an arbitrary shuffled direction.

## Outputs

- `outputs/metric_ablation_scores.csv`: ablation scores with shuffled surprisal controls.
- `outputs/prompt_features_ablation.csv`: prompt-level features used in the shuffled control.
