# Control: Fisher Output Energy

## Purpose

Compare steering directions at matched Fisher-output movement, `||F^{1/2} J delta z||`. This addresses the objection that accessible steering works only because it creates a larger output perturbation.

## Outputs

- `outputs/equal_output_energy_contrasts.csv`: accessible-versus-control contrasts at matched Fisher-output energy.
